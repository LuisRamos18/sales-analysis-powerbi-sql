# 📊 Análisis de Ventas con SQL y Power BI

## 📌 Descripción
Este proyecto presenta un análisis de ventas a partir de datos extraídos de un sistema ERP (CONTPAQi Comercial Premium), con el objetivo de generar insights para la toma de decisiones comerciales.

> ⚠️ Los datos han sido anonimizados para proteger información sensible.

---

## 🎯 Objetivo
Analizar el desempeño de ventas, identificar patrones de comportamiento y detectar oportunidades de mejora en clientes, productos y facturación.

---

## ⚙️ Tecnologías utilizadas
- SQL Server
- Power BI
- DAX
- Modelado estrella
